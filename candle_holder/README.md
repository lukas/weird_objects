# Candle holder

A spiral-shadow candle holder: the wall is a swept helical lattice that
casts a rotating shadow pattern on the table below as the flame
flickers.

## Files

| File | Purpose |
|---|---|
| `candle_holder.py` | Generates the holder STL. Run via `./run.sh candle_holder/candle_holder.py`. |
| `stl/`             | (gitignored) Generated `spiral_shadow_candle_holder.stl` / `.ply`. Rebuild via the script. |

## Build

```bash
./run.sh candle_holder/candle_holder.py
```
